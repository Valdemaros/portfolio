# Структура сайта apavlyut.ru

## Секции главной страницы

| # | Секция | Описание |
|---|--------|----------|
| 1 | **Navbar** | Навигация, имя, ссылки |
| 2 | **Hero** | Большой блок с заголовком |
| 3 | **Для кого** | Целевая аудитория |
| 4 | **Категории** | Теги типов проектов |
| 5 | **Проекты** | Сетка карточек |
| 6 | **Технологии** | Логотипы |
| 7 | **Footer** | Контакты, копирайт |

---

## 1. Navbar

```
+------------------------------------------------------------------+
| [Аватар] Имя Фамилия          GUIDE | УСЛУГИ | ПРОЕКТЫ | ...    |
+------------------------------------------------------------------+
```

- Тёмный фон (`bg-stone-900`)
- Слева: аватар + имя + подпись
- Справа: ссылки-кнопки
- Кнопка "ПРОЕКТЫ" с красной рамкой

---

## 3. Для кого

```
+------------------------------------------------------------------+
|    Для кого: бизнесмена, предпринимателя, директора, руководителя    |
+------------------------------------------------------------------+
```

- Светлый фон (`bg-stone-50`)
- Курсивный текст
- Подчёркнутые слова

---

## 4. Категории проектов

```
+------------------------------------------------------------------+
|  Стартапы | Корпоративные сайты | Интернет-Магазины | CRM системы  |
|  Веб-Сервисы | API | ERP | И другие →                              |
+------------------------------------------------------------------+
```

- Голубой фон (`bg-sky-100`)
- Ссылки-теги
- Hover эффект

## Цветовая схема

| Элемент | Цвет Tailwind |
|---------|---------------|
| Тёмный фон | `bg-stone-900` |
| Светлый фон | `bg-stone-50` |
| Акцент | `red-700`, `red-800` |
| Текст основной | `text-stone-800` |
| Текст светлый | `text-stone-300` |
| Ссылки | `text-red-600` |

---

## Шрифты

- Основной: `font-light` (тонкий)
- Заголовки: `font-bold`
- Размеры: `text-sm`, `text-lg`, `text-xl`, `text-4xl`, `text-6xl`
