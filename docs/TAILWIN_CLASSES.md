# Справочник: Классы и методы из home.html.slim

## 1. HTML теги в Slim

| Slim | HTML | Для чего |
|------|------|----------|
| `nav` | `<nav>` | Навигация |
| `section` | `<section>` | Секция страницы |
| `footer` | `<footer>` | Подвал сайта |
| `div` | `<div>` | Блок-контейнер |
| `span` | `<span>` | Строчный элемент |
| `h1` | `<h1>` | Заголовок 1 уровня |
| `h2` | `<h2>` | Заголовок 2 уровня |
| `h3` | `<h3>` | Заголовок 3 уровня |
| `p` | `<p>` | Параграф (текст) |

---

## 2. Синтаксис Slim

| Пример | Что делает |
|--------|------------|
| `nav.bg-stone-900` | Тег nav с классом bg-stone-900 |
| `.container` | `<div class="container">` |
| `#main` | `<div id="main">` |
| `= link_to "Текст", path` | Вывести ссылку |
| `= image_tag "url"` | Вывести картинку |
| `- if condition` | Условие (без вывода) |
| `/ Комментарий` | Комментарий (не виден в HTML) |

---

## 3. Rails хелперы

### link_to — ссылка

```slim
= link_to "Текст", path
= link_to "Текст", path, class: "text-red-500"
```

| Пример | Куда ведёт |
|--------|------------|
| `root_path` | Главная страница `/` |
| `projects_path` | Список проектов `/projects` |
| `project_path(@project)` | Страница проекта `/projects/1` |

### image_tag — картинка

```slim
= image_tag "https://url.com/image.png"
= image_tag "image.png", class: "w-10 h-10"
```

## 14. Шпаргалка: Что ты использовал

### Navbar
```slim
nav.bg-stone-800.text-white.py-4
  .container.mx-auto.px-4.flex.justify-between.items-center
    = link_to root_path, class: "flex.items-center.gap-3"
      .w-10.h-10.bg-red-700.rounded-full
      div
        .text-xl Владимир
    .flex.gap-4
      = link_to "Проекты", projects_path, class: "hover:text-red-400"
```

### Hero
```slim
section.bg-stone-700.text-white.py-40
  .container.mx-auto.px-4.text-center
    h1.text-4xl.font-bold.mb-4 Веб разработка
    p.text-xl.text-stone-300 Подзаголовок
```

### Карточка
```slim
.border.rounded.overflow-hidden.hover:shadow-lg.transition
  .aspect-video.bg-stone-200
  .p-4
    h3.font-bold Название
    p.text-stone-600.text-sm.mt-1 Описание
```

### Badge (метка)
```slim
span.bg-white.rounded-full.px-3.py-1 Текст
```

### Карточка технологии
```slim
.flex.flex-col.items-center.gap-2
  .w-16.h-16.bg-white.border.rounded.p-2.flex.items-center.justify-center
    = image_tag "url", class: "w-12 h-12 object-contain"
  span.text-sm Название
```

## 15. Частые ошибки

| Ошибка | Правильно |
|--------|-----------|
| `w-50 h-50` | `w-12 h-12` (Tailwind имеет шаг 4: 8, 12, 16, 20...) |
| `bg-stone` | `bg-stone-900` (нужен оттенок) |
| `text-red` | `text-red-600` (нужен оттенок) |
| Забыть `=` перед хелпером | `= link_to` (не `- link_to`) |
